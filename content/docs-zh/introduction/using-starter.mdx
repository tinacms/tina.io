要快速设置一个新的Tina启动套件，请在命令行中运行：

```bash
npx create-tina-app@latest
```

接下来，您会被提示回答一些快速设置问题：

![](/img/docs/introduction/cli-starter.png)

一旦您的本地启动套件创建完成，要运行启动套件，请按照CLI中的说明操作：

1\. 使用 `cd <your-starter-name>` 进入项目目录。

2\. 使用您选择的包管理器的开发命令运行项目。以下是pnpm的示例。

```
pnpm run dev
```
