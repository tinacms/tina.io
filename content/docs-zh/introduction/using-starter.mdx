要快速设置一个新的Tina起始项目，请在命令行中输入：

```bash
npx create-tina-app@latest
```

接下来，您将被提示进行一些快速设置问题：

![](/img/docs/introduction/cli-starter.png)

一旦您的本地起始项目创建完成，要运行该项目，请按照CLI中的说明进行操作：

1\. 使用 `cd <your-starter-name>` 进入项目目录。

2\. 使用您选择的包管理器的开发命令运行项目。请参见下面的pnpm。

```
pnpm run dev
```
