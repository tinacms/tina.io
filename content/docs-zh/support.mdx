## 社区支持

TinaCMS 的前端是免费和开源的。支持通过我们的[社区 Discord 服务器](https://discord.com/invite/zumN63Ybpf)提供（请使用 #help 频道）。可以直接在 [TinaCMS GitHub 仓库](https://github.com/tinacms/tinacms/issues/new/choose)中提交错误/功能请求。

## 优先支持

Tina 团队为付费计划的用户提供优先的 Discord 支持。要将您的 Discord 账户链接到您的 TinaCloud 账户，您可以在 [TinaCMS Discord](https://discord.com/invite/zumN63Ybpf) 的 `#✅-verify` 频道下点击“verify”。

如果您的 TinaCloud 邮箱与您的 Discord 账户不同，请点击“使用其他邮箱”按钮并验证您的 TinaCloud 邮箱。

![](/img/discord-support.png)

## 企业支持

要了解更多关于我们的企业支持计划的信息，请查看我们的[企业页面](https://tina.io/enterprise/)。

## 账户咨询

对于任何与账户相关的咨询——包括组织删除——请联系 info\@tina.io。
