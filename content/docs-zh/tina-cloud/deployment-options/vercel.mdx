## 构建配置

在Vercel中，您可以在 **Settings** > **General** > **Build & Development Settings** 更新您的构建配置。

![Vercel构建设置](/img/docs/tina-cloud-deployment-options/vercel-build-settings_vubza8.png 'Vercel构建设置')

> 如果您的package.json中有一个类似于 `tinacms build && <your-site-build-cmd>` 的“build”脚本，这可能不需要更改。如果您的Vercel配置没有运行自定义构建脚本（例如 `next build` 而不是 `npm run build`），您需要将其更改为 `tinacms build && next build`

## 环境变量

假设您的Tina `clientID` 和 `token` 已设置为环境变量，您需要将这些添加到Vercel项目的“Environment Variables”选项卡中。
