TinaCMS 是一个开源的、基于 Git 的无头内容管理系统 (CMS)——它让你可以在网站之上创建一个零代码的编辑体验。

Tina 不会将你锁定在一个专有数据库中。你的内容以 Markdown、MDX 和 JSON 文件的形式存储，并由 Git 提供支持。

继续阅读以快速浏览 **TinaCMS 的两分钟概览**，以及它如何与您的网站集成。

> 如果你想直接开始，请前往我们的 [快速入门页面](/docs/setup-overview/)。
