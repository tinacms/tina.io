TinaCMS 是一个开源的、基于 Git 的无头内容管理系统（CMS） —— 它允许您在网站之上构建一个零代码的编辑体验。

Tina 不会将你锁定在一个专有数据库中。您以 Markdown、MDX 和 JSON 文件的形式管理您的内容，这些文件由 Git 的强大功能支持。

继续阅读，了解 TinaCMS 的快速 **2 分钟介绍**，以及它如何融入您的网站。

> 如果您想直接开始，请访问我们的 [快速上手页面](/zh/docs/setup-overview/)。
