当一个集合使用模板而在文档中找不到模板键时，会发生此错误。这种情况在从字段迁移到模板或在林业迁移期间很常见。

## 如何修复

可以通过在每个出现此错误的文档中添加一个根级别的\_template字段来修复此错误。目前没有任何方法可以自动迁移内容，必须手动完成或使用自定义脚本。

例如：
如果你有一个Markdown文件，你需要添加：

```markdown
---
_template: templateName
---
```

其中 `templateName` 是你集合中模板的名称

如果是一个JSON文件，它看起来像这样。

```json
{
  "_template": "templateName",
}
```

等等。
