无需过多设置，我们的项目现在是一个功能齐全的网站构建器服务的简化版本，例如 **Wordpress** 或 **Wix**。

唯一的区别是你可以控制你的数据、应用程序以及可能实现的范围。

如果你是AI的信徒，可以从诸如 [21st.dev](https://21st.dev?tab=components\&sort=recommended) 或 [v0](https://v0.dev/) 的工具中添加区块组件。或者坚持经典，使用 [Bootstrap](https://getbootstrap.com/) 构建一个完全模块化的网站。

我们是一个小而专注的开发团队——如果你需要任何支持，请在 [discord](https://discord.com/invite/zumN63Ybpf) 上联系我们，或者从上方导航栏预约演示。
