> 我们有专门的[入门套件](/docs/introduction/using-starter)可以更快地启动和运行，但在这里我们将介绍如何将CMS添加到现有项目中。

现在我们准备开始添加Tina了！

### 安装 TinaCMS

在你的项目目录中，通过运行以下命令安装Tina…

```bash
npx @tinacms/cli@latest init
```

1. 选择 NextJS 作为框架。
2. 选择 NPM 作为你的包管理器。
3. 对 TypeScript 选择“是”。

*在 TinaCMS 添加到你的网站时，喝杯咖啡 ☕。*

> 🪟 **仅限 Windows** – 在 `package.json` 文件中更新你的开发脚本为 `"tinacms dev -c  \"next dev\""`。

### 测试应用程序

1. 运行 `npm run dev` 来启动你的新 **Tina化** 的 NextJS 应用程序!![](/img/tutorials/SCR-20250319-jkaq.png)
2. 通过访问 `http://localhost:3000/admin/index.html` 来进入你的新 Tina 编辑器

  body={<>
    **删除** `src/pages/` 目录，这是为了确保与旧版本兼容而生成的。
  </>}
