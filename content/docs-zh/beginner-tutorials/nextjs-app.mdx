> 我们有专门的[入门套件](/docs/introduction/using-starter)，可以更快地启动和运行，但在这里我们将介绍如何将CMS添加到现有项目中。

## 第一步...

1. 安装Node.js（查看[NextJS网站](https://nextjs.org/docs/app/getting-started/installation)以获取所需版本）
2. 运行 `npx create-next-app@latest`
3. 当被问到 `Would you like your code inside a src/ directory?` 时，选择 `no` – 示例代码假设如此
4. 切换到你的应用程序目录
5. 运行 `npm run dev` 来查看你的NextJS应用程序！

## 为什么选择NextJS？

[NextJS](https://nextjs.org/) 是一个行业领先的React框架，使构建快速、可扩展的网站变得容易。

我们选择NextJS作为本教程的框架，因为我们无法涵盖所有类似的框架——但TinaCMS也兼容Astro、Remix等！🚀

我们喜欢在TinaCMS项目中使用NextJS。

* **混合渲染** – 支持静态生成和服务器端渲染。
* **性能** – 自动代码拆分、图像优化和缓存。
* **SEO友好** – 预渲染页面有助于搜索引擎索引内容。

Next.js与TinaCMS配合良好，使得在保持网站快速现代的同时，轻松编辑内容。

如果你是NextJS或前端网页开发的新手，我们建议查看[他们的初学者教程](https://nextjs.org/learn)。
