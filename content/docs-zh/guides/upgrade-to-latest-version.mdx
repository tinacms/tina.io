要查看您当前使用的TinaCMS版本，请检查您的package.json文件：

```json
"dependencies": {
  "@tinacms/cli": "^1.x.x",
  "tinacms": "^1.x.x"
}
```

🔍 如何查找可用版本

> 注意：始终同时更新tinacms和@tinacms/cli到最新版本，以获取最新功能并避免兼容性问题。

您可以在npmjs.com上查看所有已发布的TinaCMS包版本：

[tinacms on npm](https://www.npmjs.com/package/tinacms)

[@tinacms/cli on npm](https://www.npmjs.com/package/@tinacms/cli)

使用您喜欢的包管理器将TinaCMS升级到最新版本：

使用npm：

```bash
npm install tinacms @tinacms/cli@latest
```

使用pnpm：

```bash
pnpm add tinacms @tinacms/cli@latest
```

使用yarn：

```bash
yarn add tinacms @tinacms/cli@latest
```

使用npm安装特定版本的TinaCMS示例

使用npm：

```bash
npm install tinacms@2.8.0 @tinacms/cli@1.10.0
```
