> 对于更高级的分支和拉取请求工作流程，请查看TinaCloud的[编辑工作流程](https://tina.io/editorial-workflow/)（仅适用于商业和企业计划）。

如果您的内容编辑需要在多个分支上工作，您可以启用Tina的分支切换插件。这允许您从Tina的UI中创建新分支并在它们之间切换。未来，我们将在此基础上构建，支持创建拉取请求、合并和其他相关工作流程。
<WebmEmbed embedSrc="/img/docs/tina-cloud/branch-selector_e5ndeg.webm"/>

## 安装

只需在您的Tina[配置文件](/docs/reference/config/)中的CMS回调函数中添加分支切换标志。如果您的站点尚未使用CMS回调函数，请将其添加到您的配置中。

```javascript
cmsCallback: (cms) => {
  cms.flags.set('branch-switcher', true)
  return cms
}
```

\
有关示例，请参见[此仓库](https://github.com/tinacms/demo-incremental/blob/main/.tina/config.tsx#L16)。
