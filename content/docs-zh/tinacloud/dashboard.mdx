**TinaCloud 仪表板**是用于管理内容管理的行政方面的界面。仪表板允许您创建**项目**，并设置您的**用户**。

![tina-cloud-dashboard](/img/docs/tinacloud/overview.png)

一个**项目**连接到您网站的 GitHub 仓库，并授权 TinaCloud 直接从中推送和拉取内容。您还可以授权**协作者**编辑您的网站。一旦通过用户管理页面添加了编辑者，在项目的**协作者标签**中授予他们访问权限。现在，他们将能够直接登录到您的网站，开始编辑内容，并将他们的更改保存到您的 GitHub 仓库。

这里一个重要的区别是，与传统的 CMS 仪表板不同，TinaCloud 的仪表板并不是用来编辑您的内容的。相反，内容编辑功能是由您，开发者，使用 TinaCMS 工具包设置的。
