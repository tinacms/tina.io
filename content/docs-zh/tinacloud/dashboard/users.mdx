项目的**协作**选项卡允许用户管理一个项目。

项目协作者可以拥有以下项目级别的角色之一：

- 编辑者
- 管理员

以下角色/权限适用于“项目管理员”和“项目编辑者”：

| 操作                        | 项目管理员 | 项目编辑者 |
| --------------------------- | ---------- | ---------- |
| 编辑项目内容                | ✔️         | ✔️         |
| 查看项目概览                | ✔️         | ✔️         |
| 管理项目配置                | ✔️         | ❌         |
| 管理项目协作者              | ✔️         | ❌         |

管理员和编辑者都被授权访问Tina配置站点的编辑路径。两种用户都可以使用TinaCMS将内容更改直接保存到站点的GitHub仓库中。

> 当站点配置为使用TinaCloud时，授权用户将被提示输入他们的TinaCloud凭据以访问站点的编辑路径。
