TinaCMS 提供了多种预构建组件（**插件**），可以通过 `ui.component` 属性使用。

将该属性设置为插件名称的字符串。

**这只会修改编辑器** – 存储的底层数据取决于使用的字段类型。

```javascript
{
  type: 'string',
  label: '背景颜色',
  name: 'color',
  ui: {
    component: "color",
    colorFormat: "rgb"
  }
}
```

> 你可以通过 `ui` 传入额外的插件配置 – 例如上例中的 `colorFormat`。
