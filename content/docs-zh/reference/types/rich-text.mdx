富文本字段通过WYSIWYG界面存储**Markdown内容**，支持文本格式、链接、媒体和自定义元素（如果与mdx一起使用）。

## 换行

要在TinaCMS富文本中插入可见的换行（例如，在引用内），请在需要换行的地方按 **Shift + Enter**（软回车）。

保存后，检查Markdown。保留的换行显示为独立的 `\` 行：

```markdown
> 引用文本…
> \
> 作者姓名
```

> 此行为不是标准Markdown规范的一部分，而是TinaCMS富文本实现的特定功能。

## 表格

查看此视频，了解如何在富文本编辑器中使用表格：

<Youtube embedSrc="https://www.youtube.com/embed/DnNTaS7lYZY?si=OoLw1D-to5qiHVNF" />

> 不支持多行单元格，因为内容被序列化为Markdown。表格仅支持单行行和单元格。如果需要表示多行内容，请使用列表或单独的字段。

***

### 列对齐

> 可视化编辑器当前不支持通过工具栏设置列对齐。如果需要在表格列中对齐内容，可以切换到原始Markdown模式并编辑表头分隔行中的对齐语法。

示例 — 居中第二和第三列：

```
| 对齐方式   |  分隔符  |    示例 |
| :---        |   :---:     |       ---: |
| 左对齐        |   `:---`    |       左对齐 |
| 居中      |   `:---:`   |     居中 |
| 右对齐       |   `---:`    |      右对齐 |
```

要了解更多关于对齐语法的信息，请参考[Markdown扩展语法](https://www.markdownguide.org/extended-syntax/#alignment)

## 示例

Tina将根据提供的配置生成适当的组件。

### 简单字段

```javascript
{
  label: "Body",
  name: "body",
  isBody: true,
  type: "rich-text",
}
```

![](/docs/rich-text-simple.png)

### 带工具栏覆盖的简单字段

更改您想要显示的内容及其在编辑器中的显示顺序

```javascript
{
		type: "rich-text",
		name: "body",
		label: "Body",
		isBody: true,
		overrides: {
			toolbar: ["heading", "bold", "italic"],
			showFloatingToolbar: false,
		},
}
```

![](/docs/rich-text-override.png)

### 带自定义组件的字段（`mdx`）

自定义组件可以嵌入到富文本块中。

```ts
{
  label: "Body",
  name: "body",
  isBody: true,
  type: "rich-text",
  templates: [
    {
      name: "Cta",
      label: "Cta",
      fields: [{
        name: "heading",
        label: "Heading",
        type: "string"
      }
    ]}
  ]
}
```

### 带默认富文本值的集合

要为富文本字段设置默认值，应指定整个AST树，如下例所示。

> 推荐的方法是填写您的数据，然后 `console.log` CMS API返回的相关字段值。\
> \
> 然后将其复制到集合的 `defaultItem` 属性中。

```javascript
{
  label: 'Blog Posts',
  name: 'post',
  path: 'content/posts',
  defaultItem: () => {
    return {
      title: 'My New Post',
      body: {
        type: 'root',
        children: [
          {
            type: 'p',
            children: [
              {
                type: 'text',
                text: 'Default Text',
              },
            ],
          },
        ],
      },
    }
  },
  fields: [
    {
      type: 'string',
      label: 'Title',
      name: 'title',
    },
    {
      type: 'string',
      label: 'Post Body',
      name: 'body',
      isBody: true,
    },
  ],
}
```
