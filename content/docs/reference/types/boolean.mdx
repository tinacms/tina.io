A toggle input that stores `true` or `false` values.

## Type Definition

```ts
type BooleanField = {
  name: string,
  type: 'boolean',
  label?: string,
  description?: string,
  list?: boolean,
  ui?: {
    component?: FC<any> | string | null,
    parse?: (value: boolean | boolean[], name: string, field: F) => any,
    format?: (value: boolean | boolean[], name: string, field: F) => any,  
    validate?: (
      value: boolean | boolean[],
      allValues: any,
      meta: any,
      field: UIField<F, Shape>
    ): string | undefined | void
  }
}
```

This is just a reference. The generic [Field](http://localhost:3000/docs/reference/fields) type definition contains further usage information.
