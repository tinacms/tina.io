A text input field used to store short-form content, such as titles or descriptions.

For indents, embolding and other formatting, use the [rich-text field type](/docs/reference/types/rich-text) instead.

## Type Definition

```ts
type StringField = {
  name: string,
  type: 'string',
  label?: string,
  description?: string,
  list?: boolean,
  options?: (string | { value: string; label: string })[],
  isBody?: boolean,
  ui?: {
    component?: FC<any> | string | null,
    parse?: (value: string | string[], name: string, field: F) => any,
    format?: (value: string | string[], name: string, field: F) => any,  
    validate?: (
      value: string | string[],
      allValues: any,
      meta: any,
      field: UIField<F, Shape>
    ): string | undefined | void
  }
}
```

This is just a reference. The generic [Field](/docs/reference/fields) type definition contains usage information.

## Examples

### Simple field

```javascript

{
  type: 'string',
  name: 'title',
  label: 'Title'
}
```

![](/img/docs/reference/hello-world.png)

### Simple list field

Setting list: true will make the value an array

```javascript
{
  type: 'string',
  name: 'title',
  label: 'Title',
  list: true
}
```

![](/img/docs/reference/value-hello-world.png)

### Field with text area component

By default, the text field is used for strings. To use a different core field plugin, specify it with the ui.component property.

```javascript
{
  label: "Description",
  name: "description",
  type: "string",
  ui: {
    component: "textarea"
  }
}
```

![](/img/docs/reference/description-example.png)

### Field with custom component

You can [create your own components](https://tina.io/docs/extending-tina/custom-field-components/) for the TinaCMS editor to store String type content.

```javascript
{
  label: "Title",
  name: "title",
  type: "string",
  ui: {
    component: ({ input }) => {
      return (
        <div>
          <label htmlFor="title">Title: </label>
          <input {...input}></input>
        </div>
      );
    },
  },
}
```
