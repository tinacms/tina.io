An input field for storing numeric values.

## Type Definition

```ts
type NumberField = {
  name: string,
  type: 'number',
  label?: string,
  description?: string,
  list?: boolean,
  ui?: {
    component?: FC<any> | string | null,
    parse?: (value: number | number[], name: string, field: F) => any,
    format?: (value: number | number[], name: string, field: F) => any,
    validate?: (
      value: number | number[],
      allValues: any,
      meta: any,
      field: UIField<F, Shape>
    ): string | undefined | void
  }
}
```

This is just a reference. The generic [Field](/docs/reference/fields) type definition contains usage information.
