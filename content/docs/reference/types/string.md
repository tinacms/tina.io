# `string`

```ts
type StringField = {
  label: string
  name: string
  type: 'string'
  list?: boolean
  options?: (string | { value: string; label: string })[]
  /** Represents the "body" of a markdown file **/
  isBody: boolean
  /** See docs/reference/toolkit/fields for customizing the UI **/
  ui?: object
}
```
