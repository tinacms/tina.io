## First Steps...

### Install Node.js 

Check the [NextJS website ](https://nextjs.org/docs/app/getting-started/installation)for the required version

### Run the npx Command

Run the below command in your Terminal.

```bash
npx create-next-app@latest
```

When asked `Would you like your code inside a src/ directory?` 

Select `no` â€“ to keep consistent with this demo.

### Run your App

1. Change directories to your app's directory
2. Run `npm run dev` to see your NextJS app!
