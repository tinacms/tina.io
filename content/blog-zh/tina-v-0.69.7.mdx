TinaCMS 版本 `0.69.7` 带来了一些修复、改进的错误处理和整体的开发者体验改进。

## 更简单的“路由器”API

现在可以在集合的定义中配置路由映射。

```tsx
// ...
  name: 'post',
  path: 'posts',
  ui: {
    router: ({ document }) => {
      // 例如，帖子项目可以在 posts/hello-world 预览
      return `/posts/${document._sys.filename}`;
    },
  },
// ...
```

[查看上下文编辑文档](/docs/contextual-editing/overview)

## 更简单的“全局表单”API

可以在集合的定义中将集合标记为“全局”。

```tsx
// ...
name: 'post',
path: 'posts',
ui: {
  global: true
},
// ...
```

全局表单可以从任何页面进行编辑。它们通过侧边栏访问。

![全局表单侧边栏](/img/blog/0.69.7/index-json.png)

![全局表单](/img/blog/0.69.7/global-form.png)

## 事件日志 UI

侧边栏现在链接到一个“事件日志”UI，这使得调试诸如 GitHub 到 Tina 的连接中断等问题更加容易。

![事件日志 UI](/img/blog/0.69.7/event-log.png)
