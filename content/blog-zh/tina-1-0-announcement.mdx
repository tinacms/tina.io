我们很高兴地宣布，TinaCMS 已经达到 1.0 版本，并即将脱离测试阶段！

TinaCMS 在三年多前首次发布，并在这段时间内取得了很大的进步。

在 TinaCMS 的早期阶段，入门门槛较高。现在，TinaCMS 的设置比以往任何时候都更简单。通过我们[最近的更改](https://tina.io/blog/upgrading-to-iframe/)，TinaCMS 现在可以与所有框架一起使用，如 Next.js、Remix、Astro、Hugo、11ty 等！

TinaCMS 的 1.0 版本还意味着我们现在可以在[语义版本控制](https://semver.org/)下发布，以便更容易管理重大更改。

## TinaCMS 入门

要尝试 Tina 入门版：

```bash
npx create-tina-app@latest
```

对于现有网站，初始化 TinaCMS：

```bash
npx @tinacms/cli@latest init
```

您可以查看“入门”文档以获取更多信息

> 对于希望迁移到最新基于 iframe 更改的现有 Tina 网站，[请参阅我们的迁移文档](https://tina.io/blog/upgrading-to-iframe/)
