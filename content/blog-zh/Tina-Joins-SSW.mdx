我很高兴地分享，Tina 已被 [SSW](https://ssw.com.au/) 收购，这是一家澳大利亚软件开发公司，也是 Tina 的长期客户。我们的服务不会中断，用户无需采取任何行动。

在收购之前，我们的预算非常紧张，只有四名开发人员的精简团队。我们难以跟上客户和社区的需求。现在，我们有资源将 Tina 提升到一个新的水平。

SSW 的创始人 Adam Cogan 非常相信我们的内容管理方法。他的公司自 Tina 早期以来就一直在他们的网站 [SSW.com.au](https://ssw.com.au/) 上使用 TinaCMS。当他有兴趣购买 Tina 时，我问他原因，他的回答让我印象深刻。他说，**“我只是希望这个项目能永远存在”**。在与 Adam 和他的团队相处的过程中，很明显他们非常关心开源软件，并且与我们建立一个支持 Git 的世界级 CMS 的长期愿景非常一致。

SSW 已运营超过 30 年，在 4 个国家拥有超过 100 名员工。他们为我们的小团队带来了丰富的经验和资源。现在我们成为 SSW 的一部分，我们开始增加 Tina 的团队规模并加速其开发。这次收购为 Tina 提供了一个稳定的基础和资源，使其成为领先的 CMS。

我将从 CEO 转变为产品负责人。这意味着我将 100% 投入时间确保产品变得出色。现在我们正在增加开发能力，我很想听听您希望 Tina 改进的地方。如果您有产品建议或发现漏洞，请在 GitHub 上发布，支持请查看我们的社区 Discord，想打个招呼的话，可以给 Adam 和我发送电子邮件（[adamcogan@tina.io](mailto\:adamcogan@tina.io) 和 [scott@tina.io](mailto\:scott@tina.io)）。

感谢 TinaCMS 社区中多年来支持我们的每一个人。

我期待我们的下一个篇章！
